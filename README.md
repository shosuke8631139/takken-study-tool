# 宅建学習サポートツール

宅建士試験合格を目指すための、完全オフライン動作の学習Webアプリです。

## 📁 ファイル構成

```
index.html   ← これ1ファイルだけで動作します（約762KB）
```

ブラウザで `index.html` を開くだけで使えます。サーバー不要・インストール不要。

---

## ✅ 実装済み機能

### 基本学習機能
- **問題演習モード** — 780問収録（id:1〜780）＋ 難問フラグ付き80問
- **カテゴリフィルター** — 権利関係 / 法令制限 / 税・その他 / 宅建業法 / 免除
- **難易度フィルター** — 通常 / 難問のみ
- **解説表示** — 正解・不正解後に詳細解説
- **正答率ダッシュボード** — 問題ごとの正解数・不正解数・ストリーク管理

### 超本番模擬試験（6セット）
| パート | 問題数 | カラー | テーマ |
|--------|--------|--------|--------|
| P1 | 50問 | 紫 #7b1fa2 | 本番レベル全分野 |
| P2 | 50問 | 緑 #1a5c3a | 本番レベル全分野 |
| P3 | 50問 | ティール #004d40 | 本番レベル全分野 |
| P4 | 50問 | インディゴ #1a237e | 本番レベル全分野 |
| P5 | 50問 | ディープローズ #880e4f | 最難関レベル |
| P6 | 50問 | ディープオレンジ #bf360c | 最難関レベル |

- 制限時間120分タイマー付き
- 分野別成績バー表示
- 全問解説付き
- スコアをlocalStorageに保存

### 付加機能
- ⭐ **ブックマーク** — 問題をブックマーク、ブックマークモードで集中復習
- 📈 **模擬試験スコア履歴** — 全6パートのスコアを記録・グラフ表示
- ⏳ **試験日カウントダウン** — 残り日数を常時表示
- 📅 **学習カレンダー & デイリーノルマ** — 毎日の学習問題数をカレンダー表示
- ✏️ **問題メモ** — 各問題に個人メモを追記
- ❌ **間違い問題再挑戦モード** — 模擬試験の間違い問題だけを再挑戦
- 🧠 **語呂合わせ一覧** — 主要暗記事項の語呂合わせ集

---

## 💾 データ保存（localStorage）

| キー | 内容 |
|------|------|
| `takken_results` | 各問題の正解・不正解・ストリーク |
| `takken_bookmarks` | ブックマーク済み問題ID |
| `takken_mock_history` | 模擬試験スコア履歴 |
| `takken_exam_date` | 試験日（YYYY-MM-DD） |
| `takken_daily_log` | 日別学習問題数 |
| `takken_daily_goal` | 1日のノルマ問題数 |
| `takken_memos` | 問題ごとのメモ |

## 🗂️ コード構成（index.html内）

```
<head> ... スタイル定義
<body>
  ├── ヘッダー（タイマー・カウントダウン・ボタン類）
  ├── 問題表示エリア
  ├── ドロワー（カテゴリ・機能メニュー）
  ├── 各種オーバーレイ（模擬試験×6、ダッシュボード、語呂合わせ、リトライ等）
  └── <script>
       ├── 問題DB（QUESTION_DB: id:1〜780、HARD_DB: 難問80問）
       ├── 模擬試験DB（SUPER_MOCK_DB〜SUPER_MOCK_DB_6: 各50問）
       ├── 語呂合わせDB（GORO_DB）
       ├── コア学習ロジック（問題表示・採点・統計）
       ├── 模擬試験ロジック（P1〜P6 各パート独立）
       └── 付加機能ロジック（ブックマーク・メモ・カレンダー等）
```
